# Agent Instructions — TightenerDocs (Public)

This is the **public** documentation repo for Tightener. It is published to GitHub and readable by anyone.

**Start here**: [Docs/index.md](Docs/index.md) for user-facing docs, [README.md](README.md) for install instructions.

## Rule — public repo only

Never add to this repo:
- internal hostnames, VM/build-machine names, or SSH/server details
- private git remotes or references to non-public sibling repos' internals
- credentials, credential-access procedures, license/discount-code database access, cert serials or signing identifiers
- absolute local filesystem paths (e.g. `/Users/...`, `C:\Users\...`)
- internal engineering roadmaps, TODOs, regression-harness notes, or AI-agent skill maps

That kind of content belongs in the private `TightenerSecrets_glueco.de` repo instead (see its `AGENTS.md` / `CLAUDE.md` / `.github/copilot-instructions.md`, not accessible from here).

Everything in this repo should be safe and useful for an external Tightener user or script developer to read.
